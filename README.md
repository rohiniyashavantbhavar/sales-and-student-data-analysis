# Data Analyst Practice Projects

This repository contains two beginner-friendly Data Analyst projects:

## 1. Sales Data Analysis Dashboard
- Dataset: `sales_data.csv`
- Tools: Excel / Power BI
- Tasks:
  - Data cleaning
  - Pivot tables
  - Charts & KPIs
  - Interactive dashboard

## 2. Student Database Analysis (SQL)
- File: `student_analysis.sql`
- Concepts:
  - SELECT, WHERE, GROUP BY, JOIN basics
  - Aggregations
  - Top performers & attendance insights

## How to Use
1. Open `sales_data.csv` in Excel or Power BI to build dashboards.
2. Run `student_analysis.sql` in MySQL / SQLite / SQL Server.
